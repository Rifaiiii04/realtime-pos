# Realtime Point of Sale (POS) App

A Realtime Point of Sale application built with **Next.js App Router** as part of the  
**Fullstack Next.js: Realtime Point of Sale Apps** online course by WPU Course.

This project focuses on transaction flow, table management, and real-time data updates,
while maintaining a clean and scalable application structure.
